## Supported systems

- iFirma (direct API export)
- Wfirma (CSV + XML export)
- Comarch ERP / Optima (XML JPK_FA)
- Saldeo (import-ready)
- Streamsoft (CSV)
- Generic format (CSV, XML, JSON)

## How to enable

Settings → Integrations → Bookkeeping export → pick a system. Your accountant gets an email invite.
